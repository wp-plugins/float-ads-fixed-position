== Description ==

This plugin usefull for advertising on sidebar. Ads will drift vertically down to the footer, ads will disappear. The position of ads always fixed while it appear. See the demo of script here: http://logoart.vn/files/admicro.html

How to use?

Easy to use this plugin. After download and active this plugin, goto Appearance --> Widget --> find Float Ads Fixed Position and drag it to your sidebar. Fill content and configuration and goto front-end to see it.

== Installation ==

1. Download the plugin.
2. Upload it to the plugins folder of your blog.
3. Goto the goto Appearance --> Widget --> find Float Ads Fixed Position and drag it to your sidebar. Fill content and configuration and goto front-end to see it.
